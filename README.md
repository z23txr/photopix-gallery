# photoPiX - Photography Portfolio Website

A modern, responsive photography portfolio website built with HTML5, CSS3, and JavaScript (jQuery, Isotope.js, and AOS). This project serves as a professional showcase featuring dynamic photo filtering, smooth animations, and a responsive layout.

## Features

- **Responsive Hero Section**: Fullscreen banner with search functionality and call-to-action styling.
- **Dynamic Portfolio Gallery**: Filterable photo gallery powered by Isotope.js (All, Nature, Fashion, Wedding).
- **Scroll Animations**: Smooth entrance animations implemented with AOS (Animate On Scroll).
- **Testimonials Section**: Interactive member testimonials and animated thumbnail preview.
- **Responsive Footer**: Modern 3-column layout with quick links, contact information, and newsletter subscription.
- **Smooth Navigation**: Anchor-based navigation linking smoothly across different sections of the page.

## Technologies Used

- **HTML5**: Semantic web structure.
- **CSS3**: Custom styling, responsive flexbox and grid layouts, and hover transitions.
- **JavaScript & jQuery**: DOM manipulation and event handling.
- **Isotope.js**: Gallery filtering and grid layout.
- **AOS Library**: Animate On Scroll effects.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/z23txr/my-1st-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-1st-website
   ```
3. Open `index.html` directly in your web browser, or use a local development server such as VS Code Live Server.
